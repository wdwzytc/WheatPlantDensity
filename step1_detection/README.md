# WheatPlantDensity_step1_leaf_tip_detection 小麦植株密度_第一步_叶尖检测

## Overview | 项目简介

This module performs leaf tip detection on wheat images as the first step of plant density estimation.
本模块用于对小麦图像进行叶尖检测，作为植株密度估计的第一步。

## 📥 Input | 输入
Image(s)
小麦图像

## Output | 输出
Detected leaf tip coordinates or points
检测到的叶尖位置（坐标或点）

## Methods | 方法
Two methods are supported:
支持以下两种方法：
- P2PNet (recommended | 推荐)
More robust 
更稳健
- YOLOv8
